# Healthcare Admin Starter Kit

A premium branded landing page for Kori Pickle's **Healthcare Admin Student-to-Career Digital Tools** niche.

This project is designed as a beginner-friendly, career-aligned digital product positioning page for healthcare administration students, patient-access beginners, prior authorization learners, revenue cycle beginners, and future remote healthcare admin workers.

## Brand Identity

**Primary colors**

- White: `#FFFFFF`
- Tennessee Orange: `#FF8200`
- Black font color: `#000000`
- Warm gray structure: soft neutral panels and borders

**Visual system**

- High-contrast editorial serif headlines
- Clean sans-serif supporting text
- Vols Orange accent system
- White background with warm gray structural elements
- Double-ring circular nodes with soft glow
- Dotted and fading connector lines
- Structured grid with intentional asymmetry
- 40–50% whitespace for breathing room
- Premium footer with creator credit, handwritten-style signature, and minimal LinkedIn/GitHub icons

## Core Niche

**Healthcare Admin Student-to-Career Digital Tools**

> I create beginner-friendly digital tools for healthcare administration students, patient-access beginners, and future remote healthcare admin workers who want to understand healthcare operations, build confidence, organize their learning, and prepare for entry-level healthcare roles without needing prior work experience.

## Featured Positioning

> I help healthcare administration students and beginners understand healthcare workflows, organize their career search, and build entry-level proof through simple digital tools.

This project is positioned as **student-created beginner tools for healthcare admin learning, career prep, and simulated no-PHI portfolio building**. It does not provide medical, coding, legal, compliance, payer, clinical, or guaranteed employment/income advice.
